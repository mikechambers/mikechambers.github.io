Adobe AIR Express Install Resources

Friday, March 7, 2008

I am currently working on updating the “Adobe AIR for JavaScript Developers Pocketguide” (O’Reilly), and was looking for resources on deploying AIR applications via express install. If you are not familiar with it, express install allows you to deploy your application via a badge on your website. Not only will it install your application, but it will also install Adobe AIR if it is not on its user’s system.
