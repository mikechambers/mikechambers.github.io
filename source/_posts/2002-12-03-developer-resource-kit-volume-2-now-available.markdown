The Flash content includes:  
Flash UI Component Set 4, which includes the following components:  
  
  
*   [Rich Text Editor][3]  
    *   [Smart Combo Box][4]  
        *   [Advanced Calendar][5]  
            *   [Color Picker][6]  
                *   [Simple Menu
